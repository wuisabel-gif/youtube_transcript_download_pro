What it is

A friendly desk tool for turning videos into readable, searchable text.

Three ways to use it, one job to do. A command-line tool for scripts and servers, a web app for quick one-offs, and a Chrome extension for the video you are watching right now. Same engine, same clean output, no accounts in sight.

The CLI

One command, one tidy file.

Point TubeScribe at a URL and pick a format. It fetches the captions, cleans up the timing, and writes the file next to you. Pipe it, cron it, drop it in a Makefile. It behaves like the rest of your toolbox.

No API key, no quota, no login dance

Formats & languages

txt, srt, vtt, json. Your call.

Want plain reading text? Use txt. Subtitles for an editor? srt or vtt. Feeding another program? json with every timestamp intact. Pick a track in any ISO language the video offers, and TubeScribe writes it out exactly that way.

If the video has the language, you can have the file
Output format
txt srt vtt json
Caption language
any en es fr de pt ja ko hi ar
Bulk download

Grab the whole channel while you make coffee.

Hand TubeScribe a playlist or a channel and it works straight down the list, one transcript after another, skipping the ones with no captions. Come back to a folder full of files and a job already done.
